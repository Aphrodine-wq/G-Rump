Quick Start

1

Download and install

Open the .dmg and drag G-Rump to your Applications folder.

2

Launch G-Rump

Open G-Rump from Applications. Grant Accessibility permissions when prompted for global hotkey support.

3

Add your API keys

Go to Settings and add API keys for your preferred AI providers (Anthropic, OpenAI, OpenRouter, etc.).

4

Or build from source

Clone the repository and run:

make run
