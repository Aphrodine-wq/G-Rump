Platform

Built for serious macOS development.

G-Rump is a Swift 5.9+ and SwiftUI application for macOS 14+ with a Node.js backend for auth, credits, billing, and OpenRouter proxying. It is distributed outside the Mac App Store to support shell execution, language servers, and deep system integration.

Requirements

  • macOS 14 Sonoma or later
  • Swift 5.9+ for local builds
  • Optional API keys for preferred AI providers
